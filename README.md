 Daily Times E‑Paper Downloader
这是一个用于下载巴基斯坦《每日时报》(Daily Times) 电子报的图形界面工具。它能够自动从官方网站抓取指定日期的电子报页面（图片格式），合并生成一个便于阅读和保存的 PDF 文件。
 ✨ 主要功能特性
  图形化界面 (GUI)：使用 Tkinter 构建，操作简单直观，无需命令行知识。
  智能下载：自动解析目标日期页面，识别有效的电子报图片。
  格式转换：将所有下载的 JPG 图片页面合并为一个 PDF 文档。
  增量检测：若某页下载失败，自动停止后续尝试，节省时间。
  日志记录：界面内嵌实时日志窗口，清晰展示每一步操作结果。
  自动清理：下载完成后自动清理临时缓存文件。

 📋 环境要求与安装
1.  Python 版本：请确保您的系统已安装 Python 3.8 或更高版本。
2.  安装依赖库：使用以下命令安装运行本程序所需的所有第三方库：
    bash
    pip install requests Pillow PyPDF2 beautifulsoup4
    
     requests`：用于网络请求和下载。
     Pillow`：用于处理图片。
     PyPDF2`：用于合并图片为PDF。
     beautifulsoup4`：用于解析网页HTML。

 🚀 使用方法
1.  启动程序：
    将代码保存为daily_times_downloader.py`，然后在终端或命令行中运行：
    bash
    python daily_times_downloader.py
    
2.  操作界面：
      在“请输入日期（YYYYMMDD）”输入框中，输入您想下载的报纸日期，例如20231025`。
      也可以点击 “查看/选择日期” 按钮进行输入。
      点击 “开始下载” 按钮，程序将开始工作。
3.  查看结果：
      下载状态和进度条会实时更新。
      所有操作日志会显示在下方的“操作日志”窗口中。
      下载完成后，程序会自动打开最终存放PDF的文件夹（默认为D:\INFO\dailytimes`）。

 ⚙️ 配置说明

python

FINAL_DIR = r"D:\INFO\dailytimes"

 🔧 核心工作原理
1.  日期转换：将用户输入的YYYYMMDD` 格式转换为网站使用的DD-MM-YYYY` 格式。
2.  页面索引：访问目标日期的电子报索引页，解析并提取文章ID (`p` 值)。
3.  图片下载：根据获取的ID，按顺序（a1.jpg, a2.jpg ...）下载图片页面，直到遇到404错误或达到最大页数。
4.  PDF合并：使用Pillow` 库将所有下载成功的JPG图片合并为单个PDF文件。
5.  清理与完成：将最终PDF保存至目标目录，并清理临时文件。
